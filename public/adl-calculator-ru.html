Калькулятор ADL-контрастаbeta

Рассчитайте приблизительный ADL контраст по нескольким измерениям или постройте график по уже известным замерам и сравните несколько проекторов.

Введите известные значения контраста. Калькулятор использует степенную модель для интерполяции полной кривой ADL от On/Off (100% чёрного) до измеренных значений ANSI и/или ADL. Чем больше точек — тем точнее оценка.

Кривая ADL-контраста

Как работает оценка

Кривая интерполируется в логарифмическом пространстве через каждое введённое значение (On/Off, 1% ADL, 5% ADL, ANSI) и точно проходит через все измеренные точки. Промежутки между точками интерполируются, а хвост за последней точкой экстраполируется по локальному наклону. Результат является оценкой; реальное поведение ADL может отличаться, особенно у проекторов с агрессивной динамической диафрагмой.

Введите точные измеренные значения контраста для каждого уровня ADL. Оставьте поля пустыми, если измерение не выполнялось; график соединит только введённые точки. Добавьте несколько проекторов для сравнения.
