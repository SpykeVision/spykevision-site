ADL Contrast Calculator beta

Estimate a projector's full ADL curve from a few measurements, or plot known data points and compare multiple projectors.

Enter the known contrast measurements. The calculator uses a power-law model to interpolate the full ADL curve from On/Off (100% black) down to the measured ANSI and/or APL points. More input points = more accurate estimate.

ADL contrast curve

How the estimate works

The curve is interpolated in log space directly through every value you enter — On/Off (0% APL), 1% APL, 5% APL and ANSI (50% APL) — so it always passes exactly through your measured points. Gaps between points are interpolated and the tail beyond your last point is extrapolated from the local slope. The result is an estimate — real-world ADL behaviour can deviate, especially on projectors with aggressive dynamic iris.

Enter the exact measured contrast values for each APL level. Leave fields blank if not measured — the chart will connect only the points you provide. Add multiple projectors to compare them side by side.
